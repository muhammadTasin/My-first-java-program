# My First Java Program

A nostalgic repository marking the beginning of my Java programming journey.

## About

This repository holds my first steps into Java programming. It represents the excitement and curiosity of starting to learn a new programming language—the foundation of everything that followed.

## The Journey Begins

Everyone has a first program. This repository captures that special moment when you write your first `public static void main(String[] args)` and see your code come to life.

## What's Here

A collection of beginner Java programs that demonstrate:
- Basic Java syntax
- Console output and input
- Simple algorithms
- Foundational concepts

## Getting Started

### Prerequisites
- Java Development Kit (JDK) 8 or higher
- Command line or IDE

### Running Your First Program

```bash
javac ProgramName.java
java ProgramName
```

## The Classic Hello World

Every journey starts here:
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
